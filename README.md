# ML/DL Practice and PoCs

This repository contains Machine Learning and Deep Learning practice exercises, proof of concepts, and learning materials.

## Project Structure

- 
otebooks/ - Jupyter notebooks for data exploration and model development
- src/ - Python source code modules
- practice/ - Practice exercises from various ML/DL tutorials
- experiments/ - Experimental code and scripts
- models/ - Trained model storage
- data/ - Dataset storage
- equirements.txt - Python dependencies

## Topics Covered

### Machine Learning
- Linear Regression (Simple and Multivariate)
- Logistic Regression (Binary and Multiclass)
- Decision Trees and Random Forest
- Support Vector Machines (SVM)
- K-Means Clustering
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Principal Component Analysis (PCA)
- Cross Validation and Grid Search
- Regularization Techniques

### Deep Learning
- Neural Networks from Scratch
- Keras and TensorFlow
- Convolutional Neural Networks (CNN)
- Transfer Learning
- Data Augmentation
- Dropout and Regularization
- GPU Benchmarking

### Data Science Tools
- NumPy and Pandas
- Matplotlib and Seaborn
- Scikit-learn
- Jupyter Notebooks

## Getting Started

1. Clone the repository:
   `ash
   git clone <repository-url>
   cd ml-dl-pocs
   `

2. Install dependencies:
   `ash
   pip install -r requirements.txt
   `

3. Start Jupyter Notebook:
   `ash
   jupyter notebook
   `
